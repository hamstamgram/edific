# Reusable Templates

Boilerplate files and starter templates for common automation patterns.

## Templates

- **n8n-workflow-template.json** — Base n8n workflow structure
- **env.example** — Example environment variables file
- **client-onboarding.md** — Client intake and onboarding checklist
- **proposal-template.md** — Project proposal template
- **invoice-template.md** — Invoice template for automation services

> Start every new project from a template to maintain consistency and save time.
