# Demos and Portfolio

This directory contains walkthroughs, screenshots, and case studies of automations built by Edific.

## Structure

    demos/
    ├── chatbot-demo/          # AI chatbot conversation samples
    ├── voice-agent-demo/      # Voice agent call recordings and transcripts
    ├── workflow-screenshots/  # n8n workflow visual exports
    └── case-studies/          # Client success stories (anonymized)

## Adding a Demo

1. Create a new subdirectory with a descriptive name
2. Include a README.md explaining what the demo shows
3. Add screenshots, recordings, or transcript files
4. Link from your portfolio page

> Demo assets are essential for client conversations. Aim to have at least one demo per service category.
